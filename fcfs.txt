Algorithm: First Come First Serve (FCFS) with Gantt Chart

Step 1: Start

Step 2: Read the number of processes (n)

Step 3: For each process, read:
        a) Process ID (PID)
        b) Arrival Time (AT)
        c) Burst Time (BT)

Step 4: Arrange all processes in ascending order of Arrival Time (AT)

Step 5: Initialize:
        current_time = 0
        total_wt = 0
        total_tat = 0
        gantt_index = 0

Step 6: For each process i from 0 to n-1:

        If current_time < AT[i] then:
            Gantt_Block[gantt_index].id = -1
            Gantt_Block[gantt_index].start = current_time
            Gantt_Block[gantt_index].end = AT[i]
            gantt_index = gantt_index + 1
            current_time = AT[i]

        Gantt_Block[gantt_index].id = PID[i]
        Gantt_Block[gantt_index].start = current_time
        
        Completion Time (CT[i]) = current_time + BT[i]
        
        Gantt_Block[gantt_index].end = CT[i]
        gantt_index = gantt_index + 1

        Turnaround Time (TAT[i]) = CT[i] - AT[i]
        Waiting Time (WT[i]) = TAT[i] - BT[i]

        current_time = CT[i]
        total_wt = total_wt + WT[i]
        total_tat = total_tat + TAT[i]

Step 7: Compute:
        Average Waiting Time = total_wt / n
        Average Turnaround Time = total_tat / n

Step 8: Display the process execution table:
        PID, AT, BT, CT, TAT, WT

Step 9: Display the Gantt Chart:
        a) Print top timeline bar: Loop through Gantt_Block from 0 to gantt_index-1. If id == -1, print "| IDLE ", else print "| P[id] ". End loop and print "|".
        b) Print bottom time markers: Loop through Gantt_Block from 0 to gantt_index-1. Print start time aligned under the block. After the loop ends, print the final end time of the last block.

Step 10: Display Average Waiting Time and Average Turnaround Time

Step 11: Stop
