# Operating Systems Lab – C Programs

This repository contains **C implementations of major Operating System lab programs** commonly required in engineering OS labs.

The programs are written for **learning, practice, and quick revision** before lab exams or viva. Students can use this repository to **review the logic, understand scheduling algorithms, and run the programs locally**.

---

## 📂 Programs Included

### CPU Scheduling Algorithms
- First Come First Serve (**FCFS**)
- Shortest Job First (**SJF – Non Preemptive**)
- Shortest Remaining Time First (**SRTF – Preemptive SJF**)
- **Round Robin Scheduling**

### Process Synchronization
- **Producer – Consumer Problem**
- **Reader – Writer Problem**
- **Dining Philosophers Problem**


### Inter Process Communication
- **Pipe Communication**
- **Shared Memory**
- **Message Passing**

### Deadlock
- **Banker's Algorithm**

### Multithreading
- **Thread-based programs using `pthread`**
  - Mean
  - Median
  - Standard Deviation
