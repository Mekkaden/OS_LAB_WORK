# 💻 Operating Systems Lab – C Programs

Welcome to the ultimate repository for **C implementations of major Operating System lab programs**. If you are tired of messy, unreadable, and overly complicated code, you are in the right place. 

These programs aren't just functional; they are written to be the **cleanest, easiest to understand, and most elegant** implementations you'll find for engineering OS labs. Built for rapid learning, smooth practice, and zero-headache revision before your finals or vivas.

Read the logic, run the code locally, and PASS the lab.

---

## 📂 Programs Included

### 🧠 CPU Scheduling Algorithms
- First Come First Serve (**FCFS**)
- Shortest Job First (**SJF – Non-Preemptive**)
- Shortest Remaining Time First (**SRTF – Preemptive SJF**)
- **Round Robin Scheduling**

### 💾 Page Replacement Algorithms
- First In First Out (**FIFO**)
- Least Recently Used (**LRU**)
- Least Frequently Used (**LFU**)

### 💽 Disk Scheduling Algorithms
- **FCFS**
- **SSTF** (Shortest Seek Time First)
- **SCAN** & **C-SCAN**
- **LOOK** & **C-LOOK**

### 🔄 Process Synchronization
- **Producer – Consumer Problem**
- **Reader – Writer Problem**
- **Dining Philosophers Problem**

### 📡 Inter-Process Communication (IPC)
- **Pipe Communication**
- **Shared Memory**
- **Message Passing**

### 🚦 Deadlock Handling
- **Banker's Algorithm** (Coded the right way—handling allocation and available resources flawlessly).

### 🧵 Multithreading
- **Thread-based programs using `pthread`**
  - Mean
  - Median
  - Standard Deviation
